# JOG ELECTROBUILD Corp. Website

Static website for an electrical and automation engineering firm.

## Project Structure
- `index.html`: Main landing page using Tailwind CSS.
- `attached_assets/`: Original content files.

## Hosting on GitHub Pages
1. Push this repository to a GitHub account.
2. Go to Repository Settings > Pages.
3. Select the branch and folder (root) to serve from.
4. The site will be available at `https://<username>.github.io/<repo-name>/`.
